HTML-The Foundation of the Web

Posted by Shanerick

HTML(Hypertext Markup Language)is the most basic building block of the web.It defines the meaning and the structure of the web content.Other technologies besides HTML are generally used to describe a web page's appearance presentation (CSS) or functionality behavior (JavaScript)

"Hypertext"refer to links that connect web pages to one another.either within a single website or between websites.Link are a fundamentals aspect of the web.By uploading content to the Internet and linking it to pages crewted by other people.you become an active participants in the World Wide Web.

What is HTML?

HTML stand for Hyper Text Markup Language.It is a markup language that defines the structure of your content.HTML consist of a series of elemants.which you use to enclose,or wrap,different parts of the content to make it appear a certain way,or act a certain way.The enclosing tags can make a word or image hyperlink to somewhere else,can italicize words,can make the font bigger or smalle,and so on.

HTML consist of elements that describe different of content:paragraph.link.heading.image.video.etc.Web browser understand HTML andrender HTML code as websites.

The Main Parts of Elements are as Follows

  1. The opening tag
  2. The closing tag
  3. The content
  4. The elements

  • Career Opportunities
  • Business Solution
